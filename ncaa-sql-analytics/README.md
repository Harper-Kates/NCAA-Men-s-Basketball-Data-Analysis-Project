## NCAA Division I Basketball SQL Analytics

### Overview
This project analyzes NCAA Division I men’s basketball team data
using MySQL to evaluate offensive efficiency and its relationship
to team success.

### Tools
- MySQL
- MySQL Workbench
- CSV data source (Sports-Reference)

### Key Metrics
- True Shooting Percentage
- True Shooting Margin vs League Average
- Win Percentage
- Points For vs. Points Against (Point Margin)

### Example Questions Answered
- Which teams shoot efficiently but still lose?
- Which teams shoot inefficiently but still win?
- Which teams score more points than their opponents but still lose more often?
- Which teams score less points than their opponents but still win more often?

### Skills Demonstrated
- SQL schema design
- Data cleaning
- CTEs and window functions
- Analytical query writing
